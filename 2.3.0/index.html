Boxmodell

CSS-boxmodell är i huvudsak en ruta som lindas runt varje HTML-element. Den består av: marginaler (margin), gränser (border), "paddning" (padding) och det faktiska innehållet (content).
Boxmodellen tillåter oss att lägga till en gräns runt elementen och definiera utrymme mellan elementen. Begreppet definieras med de fyra nämnda delarna: content, padding, border och margin.

Content

Innehållet i rutan, där text och bilder visas.

Padding

Ett utvidgare transparent område som går runt innehållet.

Border (kantlinjer)

En kant som går runt det utvidgade transparenta området (padding) och innehållet.

Margin

Ett utvigande transparent område som går utanför gränsen.

Marginaler

En marginal är en transparent rymd som går att utvdigas på ett element och som befinner sig utanför gränsområdet (border), det är den yttersta området i box-model.

margin-[ top / left / bottom / right ]

Egenskapen specificerar margin på en specifik sida (ovandsida, högersida, undersida eller vänstersida). 
          CSS (exempel)
          
            div {

              margin-top: 8px;

              margin-left: 2px;

              margin-bottom: 8px;

              margin-right: 2px;

            }

margin

Egenskapen är kort för margin-top, margin-right, margin-bottom och margin-left.
margin: length/top | right | bottom | left | initial | inherit; 
          CSS
          
            .demo-margin {

              display: inline-block;

              background: #ccc;

            }


            .demo-margin > div {

              margin: 12px;

              background: white;

            }
          
        
          HTML
          
            <div class="demo-margin">

              <div>Element med en margin på 8px</div>

            </div>
Resultat
Det yttre elementet har en grå bakgrund vilket framhävs när en margin sätts i det inre elementet.
Element med en margin på 8px

Border (kantlinjer)

En border är en rymd på ett element som definierar gränsområdet och som kan utvidgas, den ligger omkring innehållet (content).

border-[ top / left / bottom / right ]

Egenskapen specificerar bredden på en specific kantlinje (ovandsida, högersida, undersida eller vänstersida). 
          CSS (exempel)
          
            div {

              border-top: 1px;

              border-left: 1px;

              border-bottom: 1px;

              border-right: 1px;

            }

border-color

Egenskapen specificerar färgen på en eller fler kantlinjer (ovandsida, högersida, undersida och vänstersida). Om endast ett värde sätts så delegeras värdet till alla sidor. 
          CSS (exempel)
          
            div {

              border-width: black;

            }


            p {

              border-width: red green blue transparent;

            }

border-style

Egenskapen specificerar stilen på en eller fler kantlinjer (ovandsida, högersida, undersida och vänstersida). Om endast ett värde sätts så delegeras värdet till alla sidor. 
          CSS (exempel)
          
            div {

              border-style: dashed;

            }


            p {

              border-width: solid dashed solid dotted;

            }

border-width

Egenskapen specificerar bredden på en eller fler kantlinjer (ovandsida, högersida, undersida och vänstersida). Om endast ett värde sätts så delegeras värdet till alla sidor. 
          CSS (exempel)
          
            div {

              border-width: 4px;

            }


            p {

              border-width: 4px 8px 4px 8px;

            }

border

Egenskapen är kort för border-width, border-style och border-color. 
          CSS
          
            .demo-border {

              border: 4px solid black;

            }
          
        
          HTML
          
            <div class="demo-border">

              Ett element med en svart fylld kantlinje på 4px

            </div>
Resultat
Ett element med en svart fylld kantlinje på 4px

Referenser (läs vidare)

Padding (paddning)

En "paddning" är en transparent rymd som går att utvdigas på ett element och som befinner innanför gränsområdet.

padding-[ top / left / bottom / right ]:

Egenskapen specificerar explicit padding på ovandsida, högersida, undersida eller vänstersida. 
          CSS
          
            div {

              padding-top: 8px;

              padding-left: 2px;

              padding-bottom: 8px;

              padding-right: 2px;

            }

padding:

Egenskapen är kort för padding-top, padding-right, padding-bottom och padding-left. Om endast ett värde sätts så delegeras värdet till alla sidor.
padding: length/top | right | bottom | left | initial | inherit; 
          CSS
          
            .demo-padding {

              padding: 12px;

            }
          
        
          HTML
          
            <div class="demo-padding">

              Element med en paddning på 12px

            </div>
Resultat
Ett element med grå bakgrund och ett innehåll som ligger innanför paddningens rymd.
Element med en paddning på 12px

Referenser (läs vidare)

https://www.w3schools.com/css/css_padding.asp

Outlines

En outline är en rymd som går att utvdigas på ett element och som befinner sig omkring gränsområdet (border) men innanför margin. Outline är inte en del av elements dimension och påverkar inte dess storlek.

outline:

Egenskapen är kort för outline-width, outline-style och outline-color. 
          CSS
          
            .demo-outline {

              outline: 2px dashed black;

              border: 4px solid #ccc;

            }
          
        
          HTML
          
            <div class="demo-outline">

              Element med en svart "dashed" outline på 2px utanför en grå border på 4px

            </div>
Resultat
Ett element med border och outline.
Element med en svart "dashed" outline på 2px utanför en grå border på 4px

https://www.w3schools.com/cssref/pr_outline.asp

Relaterade attribut och egenskaper

visibility

Egenskapen definierar huruvida ett element skall vara synligt (visible) eller inte (hidden). 

          CSS
          
            .demo-visibility {

              visibility: hidden;

              background: plum;

            }


            .demo-visibility-toggle {

              width: 100px;

              height: 100px;

              background: pink;

            }


            .demo-visibility-toggle:hover .demo-visibility {

              visibility: visible;

            }
          
        
          HTML
          
            <div class="demo-visibility-toggle">

              <span>Lägg över musen...</span><br/>

              <span class="demo-visibility">Hej världen!</span>

            </div>
Resultat
Lägg över musen...
Hej världen!

display

Egenskapen specificerar hur ett element skall placeras och hur underelement eventuellt skall placeras ut; hur elementet skall presenteras.
De vanligaste är block och inline, där block alltid positioneras på ny rad och tar upp full möjlig bredd medan inline inte placeras på ny rad och tar endast den bredd som behövs.
Det finns ett flertal olika möjliga värden som möjliggör olika placeringar och layouts. Några av dessa gör det möjligt för mer avancerade placeringar så som flex och grid. 

          CSS
          
            .demo-flex {

              display: flex;

              align-items: center;

              justify-content: center;

            }


            .demo-grid {

              display: grid;

              grid-template-columns: max-content max-content;

            }
          
        
          HTML
          
            <div>div 1 (block)</div>

            <div>div 2 (block)</div>


            <span>span 1 (inline)</span>

            <span>span 2 (inline)</span>


            <div class="demo-flex">

              <div>Centrerat element (flex)</div>

            </div>


            <div class="demo-grid">

              <div>1:1 (grid)</div>

              <div>1:2 (grid)</div>

              <div>2:1 (grid)</div>

              <div>2:2 (grid)</div>

            </div>
Resultat
div 1 (block)
div 2 (block)
span 1 (inline) span 2 (inline)
Centrerat element (flex)
1:1 (grid)
1:2 (grid)
2:1 (grid)
2:2 (grid)

overflow

Egenskapen specificierar vad som skall hända när innehåll i ett element svämmar över (går utanför bredden och höjden). I dessa lägen kan man antingen välja att visa det med värdet visible, eller dölja det med hidden, men det går även att explicit sätta en rullningslist (scrollbar) med värdet scroll som döljer innehållet men visar det allt eftersom det scrollas, och slutligen går det implicit ha en scrollbar med värdet auto som endast visar en rullningslist om den behövs. Det går även sätta huruvida en scrollbar skall vara horisontell, vertikal eller om båda skall användas (standard).

overflow: visible | hidden | scroll | auto | initial | inherit; 
          CSS
          
            .demo-overflow-scrollbar {

              width: 100px;

              height: 100px;

              border: 1px solid #888;

              overflow: scroll;

            }


            .demo-overflow-visible {

              width: 100px;

              height: 100px;

              border: 1px solid #888;

              overflow: visible;

            }
          
        
          HTML
          
            <div class="demo-overflow-scrollbar">

              Denna ruta innehåller text som inte får plats men kan scrolla med musen för att visa resterande text.

            </div>

            


            <div class="demo-overflow-visible">

              Denna ruta innehåller text som inte får plats och flödas utanför elementet.

            </div>
Resultat
Denna ruta innehåller text som inte får plats men kan scrolla med musen för att visa resterande text.
Denna ruta innehåller text som inte får plats och flödas utanför elementet.

transparens (med rgba och opacity)

En transparens eller genomskinlighet går att få till med attributet opacity samt för färger med färgenheten rgba (eller hsla) där "a" står för alpha vilket anger genomskinlighet och representeras med ett flytvärdet från 0 till 1. 

          CSS
          
            .demo-opacity {

              display: inline-block;

              opacity: 0.5;

              width: 100px;

              height: 100px;

              background: red;

            }


            .demo-rgba {

              display: inline-block;

              width: 100px;

              height: 100px;

              background: rgba(0, 255, 0, 0.5);

            }

          
        
          HTML
          
            <div class="demo-opacity"></div>

            <div class="demo-rgba"></div>
Resultat

              width: 100px;

              height: 100px;

              box-shadow: 8px 8px 16px 5px #888; 

            }

linear-gradient

Egenskapen är en CSS function som genererar en bild som består av en progressiv övergång mellan en eller fler färger.
Varje segment separeras med ett komma och varje segment består av en färg och en frivillig position som kan vara en längd eller procent, det går även att ange en grad för att få till lutningar. 

          CSS
          
            .demo-linear-gradient {

              width: 100px;

              height: 100px;

              background-image: linear-gradient(45deg, red 0%, green 50%, blue 100%);

            }
          
        
          HTML
          
            <div class="demo-linear-gradient"></div>
Resultat

radial-gradient

Egenskapen är en CSS function som genererar en bild som består av en progressiv övergång mellan en eller fler färger som strålar från en källa/position.
Varje segment separeras med ett komma och varje segment består av en färg och en frivillig position som kan vara en längd eller procent. Det finns ett flertal möjliga attribut som justerar hur bilden genereras, som vilket form (shape) som skall användas, storlek och var den skall positioneras. 

          CSS
          
            .demo-radial-gradient {

              width: 100px;

              height: 100px;

              background-image: radial-gradient(red 0%, green 50%, blue 100%);

            }
          
        
          HTML
          
            <div class="demo-radial-gradient"></div>
Resultat
