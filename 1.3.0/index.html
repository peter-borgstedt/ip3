1.3 Semantiska element för text

<mark>

Etiketten <mark> definierar markerad text. Använd taggen <mark> om du vill markera delar av din text.

Exempel

Genom att definiera mark-etiketten så markeras texten innanför elementet enligt följande <mark>text som nu markeras</mark>.

<time> och attributet datetime

Elementet <time> är avsett för program att läsa av för att enklare omvandla värdet till ett tidsformat än om det vore text. Attributet datetime används för att översätta tiden till ett maskinläsbart format så att webbläsare kan exempelvis erbjuda datum-påminnelser via användarens kalender, och sökmotorer kan lättare producera smartare sökresultat med mera.

Exempel

Datum

<time datetime="1982">Året 1982</time>

<time datetime="1982-02">Februari året 1982</time>

<time datetime="1982-02-17">Februari 17:e året 1982</time>

<time datetime="02-17">Februari 17:e vilket år som helst</time>

<time datetime="1982-W20">Februari vecka 20</time>

Tid och datum

<time datetime="2020-01-01T00:00">Tolvslaget</time>

Sist jag var ute och <time datetime="2005-06-25 15:15">fiskade</time> var det en solig sommardag.

Tid

Klockan var <time>12.30</time> när blogginlägget skapades.

<time datetime="08:00+02:00">Kl 08.00 UTC+2</time>

Varaktighet

<time datetime="P2D">Varaktighet på 2 dagar</time>

<wbr>

Taggen (Word Break Opportunity) anger var det i en text skulle vara lämpligt att lägga till en radbrytning. Bör användas när ett ord är för långt som hint till webbläsaren att bryta raden på en specifik plats när innehållet inte ryms.

Exempel
