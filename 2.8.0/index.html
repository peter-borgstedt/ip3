2.6 Tabeller och formulär

CSS for printers

Det finns ett flertal CSS egenskaper som kan användas för att styra hur en utskrift skall se ut, bl.a. @page, break-before, break-after, page-break-after, page-break-before, size, oprhans och widows.

@page

Egenskapen används för att modifiera vissa CSS egenskaper såsom margin. Denna egenskap stöds ännu inte av Safari.

https://developer.mozilla.org/en-US/docs/Web/CSS/@page

page-break-after

Egenskapen bestämmer huruvida en sidbrytning skall ske efter ett specificerat element. Denna egenskap är idag ersatt med break-after.

https://developer.mozilla.org/en-US/docs/Web/CSS/page-break-after https://developer.mozilla.org/en-US/docs/Web/CSS/break-after

page-break-before

Egenskapen bestämmer huruvida en sidbrytning skall ske före ett specificerat element. Denna egenskap är idag ersatt med break-before.

https://developer.mozilla.org/en-US/docs/Web/CSS/page-break-before https://developer.mozilla.org/en-US/docs/Web/CSS/break-before

page-break-inside

Egenskapen bestämmer huruvida en sidbrytning skall ske unuti ett specificerat element. Denna egenskap är idag ersatt med break-inside.

https://developer.mozilla.org/en-US/docs/Web/CSS/page-break-inside https://developer.mozilla.org/en-US/docs/Web/CSS/break-inside

size

Definierar storleken och riktningen på rutan som används för att representera en sida. Egenskapen deklareras inanför egenskapen @page. Det finns ett flertal möjliga sätt att definiera värdet, bl.a. med bredd och höjd och det finns fördefinierade värden så som A5, A4, A3, portrait, landscape med flera.

https://developer.mozilla.org/en-US/docs/Web/CSS/@page/size

orphans

Egenskapen anger minsta antalet rader för ett element som måste vara synligt längst ner på en sida; det minsta antal rader som måste finnas i slutet på en sida innan den får radbrytas. Egenskapen påverkar endast block-element. Om orphans sätts till 5 så betyder det att åtminstående 5 rader måste vara synliga ovanför radbrytningen.

https://developer.mozilla.org/en-US/docs/Web/CSS/orphans

widows

Egenskapen anger minsta antalet rader för ett element som måste vara synligt högst upp på en sida; det minst antal rader som måste finnas i början på en sida efter en radbrytning. Egenskapen påverkar endast block-element. Om widows sätts till 5 så betyder det att åtminstående 5 rader måste vara synliga under sidbrytningen.

https://developer.mozilla.org/en-US/docs/Web/CSS/widows
