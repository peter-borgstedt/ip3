Fontfamilj

Attributet font-family anger teckensnittet för ett element. Attributet kan innehålla flera teckensnitt som ett fallback/reträttsystem, om webbläsaren inte stöder det första teckensnittet som angivits så försöker man med nästa teckensnitt.

font-family

Vid angivelse så anges först family-name, den font man vill ha, därefter en generisk font-familj. Det går att ange flera genom att seperarera dem med komma, dessa används som fallback/reträtt om fonten inte hittas och utförs i den ordning man angivit dem. Det går att ange font-familjer med citattecken om namnet skulle bestå av mellanrum.

Attributet definieras primärt av två delar, family-name och generic-family.

font-family: family-name | generic-family | initial | inherit;

family-name

Namnet på en specifik typsnittsfamilj, som "Times New Roman", "Courier New" och "Arial" m.fl.

generic-family

Namnet på en generisk familj, "serif", "sans-serif" och "monospace" är alla en kategori med liknande font-familjer.

Sans är en generisk familj som anger font-familjer med små linjer i kanterna på vissa tecken, ex. Times New Roman.
Sans-serif där sans betyder "utan" är en generisk familj som anger font-familjer utan små linjer i kanterna på tecken, ex. Arial.
Monospace är en generisk familj som anger font-familjer där alla tecken har samma bredd, ex. Courier New.

Exempel - Font med fallback/reträttsystem

          CSS
          
            .font-family-with-fallback {

              font-family: Arial, Helvetica, sans-serif;

            }

          
        
          HTML
          
            <p class="font-family-with-fallback">

              Vald font är Arial, om den inte finns används Helvetica annars en frivillig font från generisk familj Sans-serif.

            </p>
Resultat

Vald font är Arial, om den inte finns används Helvetica annars en frivillig font från generisk familj Sans-serif.

Enheter

Det finns ett flertal olika enheter i CSS för att uttrycka storlek på, det finns absoluta och relativa enheter, nedanför ges exempel på några av de vanligaste.

Absoluta nyckelord

Nyckelord är utav en absolut storlek och baseras på användarens standardtypstorlek. 

          CSS
          
            .unit-xx-small,

            .unit-x-small,

            .unit-small,

            .unit-medium,

            .unit-large,

            .unit-x-large,

            .unit-xx-large {

              margin: 8px;

              padding: 4px;

              background-color: #eee;

              border: solid 1px #ccc; 

            }


            .unit-xx-small {

              font-size: xx-small;

            }


            .unit-x-small {

              font-size: x-small;

            }


            .unit-small {

              font-size: small;

            }


            .unit-medium {

              font-size: medium;

            }


            .unit-large {

              font-size: large;

            }


            .unit-x-large {

              font-size: x-large;

            }


            .unit-xx-large {

              font-size: xx-large;

            }
          
        
          HTML
          
            <div class="unit-xx-small">xx-small</div>

            <div class="unit-x-small">x-small</div>

            <div class="unit-small">small</div>

            <div class="unit-medium">medium</div>

            <div class="unit-large">large</div>

            <div class="unit-x-large">x-large</div>

            <div class="unit-xx-large">xx-large</div>

Procent (%)

Enhet anges i procent och är relativ till elementets storlek eller rymd.
För fonter kommer procent agera liknande som enheten em, see nedanför om dess definition.

Emphemeral (em)

Enheten definieras som relativ till typsnittets storlek. 2em betyder 2 gånger av den aktuella storleken på teckensnittet. Värdet är relativt till förälderns teckensnittstorlek, som ett resultat kommer värdet att "kaskadera" om det används på barnelement.

Egenskaper

Teckensnitt

color

Anger fontfärg; färgen på texten. En färg kan anges med ett flertal enheter som HEX, RGB, RGBA, HSL, HSLA samt med fördefinierad nyckelord, såsom "red", "green" och "blue" m.fl.

Exempel på färgen vit definierad med de olika färgenheterna:
color-white-hex {
  color: #FFFFFF;
}

color-white-rgb {
  color: rgb(255, 255, 255);
}

color-white-rgba {
  color: rgba(255, 255, 255, 1);
}

color-white-hsl {
  color: hsl(0, 0%, 100%);
}

color-white-hsla {
  color: hsla(0, 0%, 100%, 1);
}

color-white-keyword {
  color: white;
}

font

Kort egenskap för font-style, font-variant, font-weight, font-size/line-height och font-family. Om inte alla anges så kommer de värden som saknas att implicit få ett standardvärde.

font-style

Anger textstil, d.v.s. huruvida den skall vara kursiv, sne eller normal.

font-variant

Anger om en text ska visas i ett teckensnitt med små versaler.

font-weight

Anger hur tjocka eller tunna tecken ska vara.

font-size

Anger storleken på ett teckensnitt.

line-height

Anger höjden på en linje.

Referenser (läs vidare)

https://www.w3schools.com/css/css_font.asp https://www.w3schools.com/cssref/pr_font_font.asp https://www.w3schools.com/cssref/pr_font_font-variant.asp https://www.w3schools.com/cssref/pr_font_font-style.asp https://www.w3schools.com/html/html_colors.asp

Exempel: på egenskaperna "font" och "color"

          CSS
          
            .font-and-color-property {

              font: italic bold 18px Arial, sans-serif;

              color: brown;

            }

          
        
          HTML
          
            <p class="font-and-color-property">

              Denna text är kursiv, fet och har en textstorlek på 18pixlar. Vald font är Arial med sans-serif som reträtt.

            </p>
Resultat

Denna text är brun, kursiv, fet och har en textstorlek på 18pixlar. Vald font är Arial med sans-serif som reträtt.
