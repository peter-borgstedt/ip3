Hyperlänkar

Hyperlinkar eller länkar kan stylas med hjälp av de inbyggda psuedo-klasserna, vilka beskriver ett specifikt tillstånd för en länk.

a:link

Aktuell när en länk ännu inte har besökts.

a:active

Aktuell när en länk är selekterad (intryckt eller inklickad).

a:visited

Aktuell när en länk redan har besökts (efter :aktiv går stadiet över till :visited)

a:hover

Aktuell när musen är positionerad över länken.

Exempel - Länk stylad som knapp

          CSS
          
            a:link-button.link {

              display: inline-block;

              background: #f5f5f5;

              border: 1px solid #aaa;

              border-radius: 8px;

              padding: 8px;

              text-decoration: none;

            }


            a:active {

              color: greenyellow;

            }


            a:hover {

              color: #222;

            }


            a:active {

              background: goldenrod;

              color: black;

            }


            a:visited {

              background: #555;

              color: white;

              border-color: black;

            }
          
        
          HTML
          
            <a class="link-button" href="https://en.wikipedia.org/wiki/Commodore_64">

              Commodore 64

            </a>
Resultat
Commodore 64

Exempel - Länk som bild

          CSS
          
            a.link-image {

              display: inline-block;

              background: #f5f5f5;

              border: 1px solid #aaa;

              border-radius: 8px;

              padding: 8px;

            }


            a.link-image img {

              max-width: 200px;

            }


            a.link-image:link,

            a.link-image:link img {

              opacity: 0.5;

              color: greenyellow;

            }


            a.link-image:hover,

            a.link-image:hover img {

              opacity: 1;

              background: rgb(200, 220, 190);

              color: #222;

            }


            a.link-image:active,

            a.link-image:active img {

              opacity: 1;

              background: goldenrod;

              color: black;

            }


            a.link-image:visited,

            a.link-image:visited img {

              opacity: 0.5;

              background: #555;

              color: white;

              border-color: black;

            }
          
        
          HTML
          
            <a class="link-image" href="https://en.wikipedia.org/wiki/Commodore_64">

              <img src="images/c64.png" alt="https://en.wikipedia.org/wiki/Commodore_64">

            </a>
Resultat

Cursor

Attributet cursor anger vilken musikon som skall användas när användaren interagerar med element. Det finns ett flertal inbyggda ikoner som kan användas och anges som text, det finns även möjlighet att välja en egen ikon genom att ange url(http://min.bild.png).
Bytet av ikonen ger en tydlig indikation på hur det går att interagera med element utan att behöva ange beskrivande text huruvida man kan trycka, dra och rulla (scrolla).

Exempel - Inbyggd markör samt egen vald bild som markör 

          CSS
          
            .cursor-crosshair,

            .cursor-custom {

              display: inline-flex;

              width: 200px;

              height: 200px;

              border: 1px dashed #ccc;

              align-items: center;

              justify-content: center;

              color: #888;

            }


            a.cursor-drag {

              cursor: drag;

            }


            a.cursor-custom {

              cursor: url(images/c64-logo.png);

            }

  
          
        
          HTML
          
            <div class="cursor-crosshair"><div>

            <div class="cursor-custom"><div>
Resultat
Inbyggd (crosshair)
Egen (c64-logo.png)
