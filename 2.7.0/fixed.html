2.7 Sidlayout - Fixerad

Position

Denna del går igenom en del av de vanligaste egenskaperna för att positionera element.

position

Egenskapen definierar hur positioneringen skall utföras för element. Anges inget värde så används standardvärdet static, denna position har inga speciella egenskaper och positionerar enligt sidans normala flöde, det går bland annat inte sätta width, height, top, bottom, left eller right med denna positionering. Anges värdet relative så placeras elementet relativt från sin normala position. Anges värdet fixed så positioneras elementet relativt till visningsområdet (viewport) och är alltid fixerad vid sin plats även om rullning (scroll) sker. Anges värdet absolute så positioneras element relativt till närmsta positionerad ättling, hittas ingen så används <body>. Anges värdet sticky så positioneras elementet baserat på användarens rullningsposition (scroll).

top

Egenskapen påverkar den vertikala positioneringen från ovansidan och har inget effekt på element med display satt till static, skillnader kan ske hur den beter sig beroende på vald position.

left

Egenskapen påverkar den horisontella positioneringen från vänstersida och har inget effekt på element med display satt till static, skillnader kan ske hur den beter sig beroende på vald position.

bottom

Egenskapen påverkar den vertikala positioneringen från undersidan och har inget effekt på element med display satt till static, skillnader kan ske hur den beter sig beroende på vald position.

right

Egenskapen påverkar den horisontella positioneringen från högersida och har inget effekt på element med display satt till static, skillnader kan ske hur den beter sig beroende på vald position.

z-index

Egenskapen definierar djupet och huruvida elementet lägger sig över eller under andra element vilket bestäms av satt index.

float

Egenskapen möjliggör att elementet flyter från vänster eller höger sida (se även uppgift 2.6.0).

clear

Egenskapen gör att element inte kan flyta från vänster, höger eller vid båda sidorna (se även uppgift 2.6.0).

Fixerad Layout

En fixerad layout är när en sida har en yttre del som är fixerad till en specifik storlek (oftast bredd) vilket därefter omger övriga innehållet. Detta gör att vid en förändring av fönsterstörleken eller vid inzomning så vidbehåller sidan sina proportioner. De flesta element anges antingen med procent eller med en fixerad precision som px, cm eller inch.

Tabeller som layout

En tabell är inte lämplig för sidlayout eftersom de är väldigt långsamma. Anledningen till detta är att webläsaren måste ladda ner det mesta av innehållet om inte allt för att rendera den ordentligt (p.g.a. den algoritm som används med egenskapen table-layout). En tabell kräver betydligt mer element än andra sidlayouts och innebär långsammare laddning, rendering samt ökad bandbredd. En tabell kan även vara väldigt komplex att underhålla. Tabeller är även inte lika flexibla som andra sidlayouts och att få en tabell att bli responsiv kan vara väldigt svårt. Det kan även bli svårare för at sökmotorer att kategorisera innehållet från en tabell. Tabeller ger även inga bra resultat vid utskrifter.
Tabeller är inte avsedda att användas som sidlayout.

Storlek

Denna del går kort igenom en del av de vanligaste egenskaperna för att bestämma storlekar på element. Det finns ett flertal olika enheter för att bestämma storlekar så som pixel, cm, inch, procent, em och presentationen kan bete sig olika beroende på vilken enhet som väljs speciellt vid inzoomning eller förändring av fönsterstorleken.

width

Egenskapen definierar bredden på elementet.

height

Egenskapen definierar höjden på elementet.

max-width

Egenskapen definierar maximum-bredden på elementet och kan inte bli större än det valda mått som angetts.

max-height

Egenskapen definierar maximum-höjden på elementet och kan inte bli större än det valda mått som angetts.

min-width

Egenskapen definierar minimum-bredden på elementet och kan inte bli mindre än det valda mått som angetts.

min-height

Egenskapen definierar minimum-höjden på elementet och kan inte bli mindre än det valda mått som angetts.

vertical-align

Egenskapen definierar hur elementet skall placeras vertikalt. Endast inline-element stöds, d.v.s där egenskapen har display värdet inline och inline-block, även table stöds. Det går att placera element top, middle eller bottom med flera.
