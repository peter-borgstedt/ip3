Tabeller

Tabeller går att anpassa med ett flertal egna attribut, och bland de viktigare är table-layout.

Table-Layout

Attributet table-layout anger hur en tabell skall lägga ut sina rader, kolumner och celler. Det finns två möjliga lägen, fixed och auto.

auto

Tabellens bredd och dess celler justeras för att passa innehållet och en kolumns storlek bestäms av dess största cell-innehåll. För att beräkna ut bredden behövs alla celler analyseras och är därmed långsammare än en fixerad layout.

fixed

Tabellens bredd bestäms av den satta bredden på kolumnerna, eller bredden på första raden av celler. Vid en fixerad layout behövs endast första raden för att rendera övriga rader, vilket är betydligt snabbare än att använda auto.

Exempel - Fixed (och stylad)

          CSS
          
            table.table-fixed {

              table-layout: fixed;

              border-collapse: separate;

              border-spacing: 8px;

              text-align: left;

            }


            table.table-fixed,

            table.table-fixed th,

            table.table-fixed td {

              border: 1px solid #555;

            }


            table.table-fixed tr:first-child {

              text-align: center;

              background: rgb(240, 200, 250);

            }


            table.table-fixed tr:nth-child(even) {

              background: rgb(240, 230, 250);

            }


            table.table-fixed tr:first-child > th:not(:first-child) {

              width: 150px;

            }

Formulär

Formulär består huvudsakligen av själva formuläret <form>, inmatningsfält <input>, textareor <textarea>, fältuppsättning <fieldset>, kort beskrivande översikt <legend> samt flera (vilka alla går att justera för rätt stil).

Attribut selektorer

Det går via attributselektorer ange vilken typ av inmatningsfält <input> man vill styla.

input[type=text]

Ger möjlighet att styla inmatningsfält av typen text. 
          CSS
          
            input[type="text"] {

                width: 100%;

                border: padding 8px;

                border: 1px solid #888;

                border-radius: 4px;

            }
          
        
          HTML
          
            <input class="input-styled" type="text" placeholder="Stylat inmatningsfält">
Resultat

Exempel: på ett stylat formulär (och dess innehåll)

          CSS
          
            form.form-styled {

              display: flex;

              flex-direction: column;

              border: 1px solid #888;

              background: whitesmoke;

              padding: 8px;

              border-radius: 2px;

              width: 480px;

            }


            form.form-styled input[type="email"],

            form.form-styled input[type="password"] {

              width: 100%;

              padding: 8px;

              border: 1px solid #888;

              border-radius: 4px;

            }


            form.form-styled input[type="submit"] {

              width: 100px;

              padding: 4px;

              border: 1px solid #888;

              border-radius: 4px;

            }


            form.form-styled label {

              display: block;

              margin: 12px 0 4px 1px;

              font-size: 14px;

              font-weight: bold;

              color: #555;

            }


            form.form-styled legend {

              margin: 0 0 8px 0;

              text-align: center;

              font-size: 22px;

              color: #555;

            }


            form.form-styled fieldset {

              background: white;

              border-radius: 2px;

              border: 1px solid #888;

              margin: 0;

            }


            form.form-styled .button-container {

              display: flex;

              align-items: center;

              justify-content: flex-end;

              margin-top: 8px;

            }
          
        
          HTML
          
            <form class="form-styled">

              <fieldset>

                <legend>Välkommen!</legend>

                <label for="email">Epost</label>

                <input type="email" id="email" name="email" placeholder="Epost">

                <label for="password">Lösenord</label>

                <input type="password" id="password" name="password" placeholder="Lösenord">

              </fieldset>

              <div class="button-container">

                <input type="submit" value="Logga in">

              </div>

            </form>
